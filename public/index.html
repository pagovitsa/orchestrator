GitHub setup

Step 1 of 4

1. Generate the orchestrator SSH key

An ed25519 keypair lives only on this server (under /data/secrets/github/). The supervisors use it for every git push and git clone.

2. Add the public key to your GitHub account

Open github.com/settings/ssh/new, paste the key below, save with any title (e.g. "Orch UI").

3. Verify SSH access

We'll run ssh -T git@github.com with the key — GitHub answers with Hi <login>! on success.

4. Paste a Personal Access Token

Needed for creating new repos via the API (push works with SSH alone). Create one at github.com/settings/tokens with repo scope. Stored only here.

All set

Supervisors now have GIT_SSH_COMMAND, GITHUB_TOKEN, and GH_TOKEN in their environment.
